# Archive Sessions

Archive Claude Code sessions to brain knowledge base and Obsidian vault.

## Usage

- `/archive-sessions` — Archive new sessions incrementally (brain + Obsidian)
- `/archive-sessions --force` — Re-archive all sessions
- `/archive-sessions --dry-run` — Preview what would be archived
- `/archive-sessions stats` — Show archive statistics

## Handler

Invoke the `bizbrain-os:session-archiving` skill with the provided arguments.
